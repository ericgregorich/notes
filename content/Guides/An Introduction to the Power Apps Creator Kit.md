## Creator Kit
[The Creator Kit](https://learn.microsoft.com/en-us/power-platform/guidance/creator-kit/overview) is a solution that adds many additional controls to your Canvas [[Microsoft Power Apps|Power Apps]]. These controls are based on the [Fluent UI Framework](https://developer.microsoft.com/en-us/fluentui#/) and look and feel like everything else in the Microsoft ecosystem.
## Prerequisites
- Go to your [admin center Environments](https://admin.powerplatform.microsoft.com/environments).
- Select your Environment.
- Select Settings.
- Select Product > Features.
- Enable the Power Apps component framework for canvas apps.

![](d217ee49-c2d7-446a-8a29-89b54471fcf3_716x196.webp)
## Installing the Creator Kit
- Click Save.
- [Download the Creator Kit](https://aka.ms/creatorkitdownload).
- Navigate to your Power Apps Solutions page.
- Import the solution you downloaded.
## Reference Solutions
The Creator Kit also comes with two reference solutions that you can use to get familiar with the controls. You can [download them here](https://aka.ms/creatorkitdownload). Installation is the same process as above.
## Using the Creator Kit Components
- Edit your Canvas App.
- Open the Insert panel, and click “Get more components” at the bottom.

![](df185411-7d50-41bc-a717-20b3297fa927_441x937.webp)

- A pane will open on the right with the options to pick from Canvas or Code.
- Select the Code tab.
- Import the control you want to use in your Canvas App.
- A new sub-menu will appear named Code Components. It will list all the custom Creator Kit code components you have added to your app. You can now use them like any other control!

![](1743c779-5543-41b6-b72e-d01cbb59888d_518x834.webp)

## A warning appears when you edit the app.
After adding any custom components to your Canvas App, you will receive a warning when you edit the app.  It will list all of the components the app uses. You can select Open App to bypass this.

![](e241bae3-7c75-42d4-a00c-ecb0b56e3028_497x240.webp)

## What controls are included?
The full list of components and the documentation for each can be found [here](https://learn.microsoft.com/en-us/power-platform/guidance/creator-kit/components).

The most common components I use are the **Fluent Details List**, **Breadcrumb**, and **Search Box**. I use the out-of-the-box controls before entering the Creator Kit.
