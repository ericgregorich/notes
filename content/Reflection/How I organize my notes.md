## Folders
Folders represent boundaries. They don't change. A note can only be in one folder, which means the folders are very high level and have a specific purpose. 

| Folder Name | Purpose |
| ---- | ---- |
| Assets | Images and other supporting files. |
| Blog | Where I write my blog posts that are sent to Micro.blog. |
| Books | Books that I have referenced in my notes. |
| Canvas | Canvas diagrams. |
| Fleeting | An inbox where I dump new ideas and resources that need to be reviewed at a later time. |
| Guides | Instructions that are written to accomplish a specific task. |
| Newsletter | Where I write my newsletter. |
| People | People who I have referenced in my notes. |
| Quotes | Inspiring quotes. |
| Reflection | My own thoughts on various topics. |
| Topics | High-level topics that I write about. |
## Tags
I still haven't decided if I use tags. I may use them internally to track notes I want to continue to process, but I may use Obsidian Bookmarks for that instead.

## Links
I use links for people, places, and things. I also link related notes. Ideally, the link is embedded in the note with clear context given for why the link exists. 

I’ll also link to references and external resources.
